# EasyREST

## A project by Nitin Garg

* A simplified and user friendly version of PostMan
* Built for the beginners 
* Comes with advanced features such as Prettify JSON
---
### Dependency List
1. tkinter
2. ttkbootstrap
3. requests
---
### Command to use it
``` bat
pip install -r requirements.txt
```
``` bat
python main.py
```
